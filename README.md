# La-Cuillere
Application cross plateforme de création de liste de courses. (**Cordova**, **JQuery**, **SQLite**)

# Features
- Ajouter un recette
- Ajouter les ingrédients de la recette individuellement
- Affichage de la liste de courses avec incrémentation des quantités de chaque ingrédient
- **Partage** de la liste de courses par mail ou sms

Les recettes et leur composition (ingrédients) sont enregistrées dans une base **SQLite**. 

**Le projet n'est pas finalisé!**

Il manque notamment le rafraîchissement de la liste des recettes disponibles après insertion d'une nouvelle recette.
